# swarm-cli

Minimal AI terminal for coders.

## Quick Start

```bash
# Install dependencies
bun install

# Build binary
./build.sh

# Install globally
./install.sh
```

## Development

```bash
bun dev
```

## Packages

| Package | Description |
|---------|-------------|
| `packages/swarm` | CLI application |
| `packages/sdk/js` | JavaScript SDK |
| `packages/plugin` | Plugin system |
| `packages/script` | Build utilities |

## Configuration

Config file: `.swarm/swarm.json`
